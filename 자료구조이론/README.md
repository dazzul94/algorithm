# 자료구조이론

### 1. 주피터 노트북 설치
- 프로그램 설치: anaconda 설치
    + anaconda란?
        * 파이썬 기본(컴파일러)
        * 파이썬 주요 라이브러리
        * jupyter notebook 등 유용한 툴
- 만약 anaconda가 설치가 안된다면?
    + 파이썬 컴파일러 설치
    + jupyter notebook 설치

### 2. 주피터 노트북이란? 
- 마크다운, 코드
- Editor(PyCharm) VS jupyter notebook
    + 한줄 한줄 코드 실행 결과 확인이 쉽다.
    + 문서와 코드를 함께 작성/저장할 수 있다.
    > 복잡한 자료구조/알고리즘을 보다 쉽게 정리하고, 익히기 위해 사용
